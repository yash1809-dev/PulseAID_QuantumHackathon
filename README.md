# PulseAID — Quantum Hackathon

PulseAID is a real-time healthcare platform for ICU bed tracking, ambulance monitoring, and smart hospital selection using AI-driven decision support.

## Features
- **Smart Decision Engine**: Multi-dimensional hospital matching (6 analyzers)
- **Real-time Tracking**: Live ICU bed availability and ambulance locations
- **Hospital Portal**: Manage beds, doctors, and incoming patients
- **Patient Ecosystem**: Insurance/Scheme integration and medical history matching

## Setup
1. Clone the repository
2. Run `npm install`
3. Copy `.env.example` to `.env` (already provided in repo for demo)
4. Run `npm run dev`
